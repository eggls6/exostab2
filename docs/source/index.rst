Welcome to Exostab 2.0 documentation!
===================================

**Exostab** (/exo'stab/) is an online service intended to simplify dynamical stability assessment for extrasolar planets.
It pulls data from a large number of numerical, high precision orbit simulation results  
and offers a *simple* and *intuitive* API.

Check out the :doc:`API` section for further information.

.. note::

   This project is under active development.
